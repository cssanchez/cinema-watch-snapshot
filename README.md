# Cinema Watch Snapshot

This repository is the public static snapshot for Cinema Watch.

It is generated from a separate private scraper/workbench repo and published as a read-only site for sharing.

What lives here:

- the latest exported static site under `docs/`
- no live scraping logic
- no private working files

GitHub Pages should be configured to serve from:

- branch `main`
- folder `/docs`

The site updates whenever a new snapshot is exported and pushed from the private source repo.
